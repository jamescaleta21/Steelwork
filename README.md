# Steelwork

Aplicación de escritorio desarrollada en Visual Basic 6 para [Fabrica de MEgales].

## Estructura del proyecto

- `*.vbp`: Archivo principal del proyecto VB6.
- `*.frm`: Formularios del proyecto.
- `*.bas`: Módulos estándar.
- `*.cls`: Clases.
- `bin/`: Archivos compilados (`.exe`, `.dll`).
- `docs/`: Documentación del sistema (manual de usuario, etc.).

## Requisitos

- Visual Basic 6.0 (IDE)
- Sistema operativo Windows XP, 7, 10 o superior
- Librerías adicionales si las hay (ej. MSADODC.OCX, COMCTL32.OCX)

## Compilación

1. Abrir el archivo `.vbp` con el IDE de Visual Basic 6.
2. Revisar referencias (Menú: `Project` ? `References...`).
3. Desde el menú `File`, seleccionar `Make <AppName>.exe`.
4. Guardar el ejecutable en la carpeta `bin/`.

## Uso

1. Ejecutar `MiAplicacionVB6.exe` desde la carpeta `bin/`.
2. Seguir las instrucciones en pantalla o revisar el manual en `docs/ManualUsuario.pdf`.
